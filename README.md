Este repositório documenta minha jornada prática de aprendizado em AWS, Análise de Dados e Engenharia de Dados.

O objetivo é aprender através de projetos reais, documentando arquitetura, implementação, erros encontrados, correções aplicadas e aprendizados adquiridos durante os laboratórios.

---

# Projetos

## 01 - Processamento de Avaliações com Amazon EMR Serverless

Status: ✅ Concluído

### Tecnologias Utilizadas

* Amazon S3
* IAM
* Amazon EMR Serverless
* Apache Spark

### O que foi realizado

* Criação de bucket Amazon S3
* Upload de dataset e script Spark
* Configuração de permissões IAM
* Criação de aplicação EMR Serverless
* Execução de Job Spark
* Correção de erro AccessDenied
* Geração de resultados no Amazon S3

### Principais Aprendizados

* Como o Amazon S3 funciona como camada de armazenamento
* Como IAM controla permissões entre serviços
* Como Roles e Policies funcionam
* Como executar aplicações Spark sem gerenciar servidores
* Como investigar e corrigir erros de permissão
